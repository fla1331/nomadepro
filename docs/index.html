Getting started

Quick start guide for Nomade Pro

Installation

Install via npm or include the script:

// npm
npm install nomadepro

// CDN
<script src="https://cdn.example.com/nomadepro.min.js"></script>

Usage

Basic example to initialize the library:

<script>
    import NomadePro from 'nomadepro'

    const app = new NomadePro({ container: '#app' })
    app.start()
</script>

API

Common methods and options:

new NomadePro(options)
.start()
.stop()
.on('event', handler)
